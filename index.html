Summary: Weather Pi

This project answers the question "What's the weather like as we approach the equater?" Python script is utilized to visualize the weather from 500+ cities across the world. Different types of scatterplots will showcase the relationship of:

  • Temperature (F) vs. Latitude
  • Humidity(%) vs. Latitude
  • Cloudiness (%) vs. Latitude
  • Wind Speed (mph) vs. Latitude
